Welcome to the Sprite Batch Tutorial!
Prereq: Intro Graphics

Sprite batching allows us to draw textures to the screen
without a lot of difficulty. In this tutorial, after making
a sprite batch, we can use teh spirte batch to draw hundreds
of sprites to the screen. Sprite batches can be found pre-built
in libraries like MonoGame, this tutorial shows how to make one
yourself
